# 리플랙션

## 자바는 **동적 로딩(Dynamic Loading)** 을 지원한다.      
자바 프로그램은 여러개의 클래스로 구성되어 있다.             
자바는 프로그램 실행 시에 모든 클래스를 로딩하지 않고              
**필요한 시점에 필요한 클래스를 로딩**하여 사용하는 **동적 로딩(Dynamic Loading)** 을 지원한다.
